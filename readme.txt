== Description ==

This plugin ads the new Wordpress Media Library under the posting box. The plugin allows the poster uploading and posting at the same time.

Many people don't like the LightBox because it limit the dynamic and functionality of the posting so this plugin offers the user a nonlight box media library


== Installation ==

You can enjoy this plugin following the instructions:

1. Upload `wp_media_library.php` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. It will display a new box under the Write box

== Frequently Asked Questions ==

= Why it desapears after sending the information to the post? =

Because it's using the default library Wordpress shows in the lightbox, however you can bring it up again by clicking "refresh" link.
